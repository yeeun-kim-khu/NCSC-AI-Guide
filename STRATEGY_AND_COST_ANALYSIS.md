## 1️⃣ **재방문율 향상 전략: 과학관 방문 없이도 사용 가능한 문제 해결**

### ❌ **문제점**
- 오디오북 기능이 집에서도 사용 가능 → 과학관 재방문 동기 감소
- 앱의 유익함 ↑ but 과학관 방문율 ↓

### ✅ **해결 전략: "방문 인증 시스템"**

#### **A. QR 코드 기반 방문 인증**
```
과학관 방문 → 각 놀이터 QR 코드 스캔 → 인증 완료 → 오디오북 생성 가능
```

**구현 방법:**
1. **각 놀이터에 고유 QR 코드 설치**
   - AI놀이터, 탐구놀이터, 관찰놀이터, 행동놀이터 각각 QR
   - QR 코드는 당일만 유효한 토큰 생성

2. **앱에서 QR 스캔 기능 추가**
   - 스캔 시 방문 기록 저장 (날짜, 시간, 놀이터)
   - 서버에 방문 로그 전송

3. **오디오북 생성 조건**
   - 최소 2개 이상의 놀이터 QR 스캔 필요
   - 당일 방문 인증만 유효 (24시간 제한)
   - 인증 없이는 "샘플 오디오북"만 제공 (30초 미리듣기)

#### **B. 위치 기반 인증 (GPS)**
```
과학관 반경 100m 내 → 앱 활성화 → 전체 기능 사용 가능
과학관 외부 → 제한된 기능만 (퀴즈, 일반 안내)
```

**장점:**
- QR 코드 설치 불필요
- 자동 인증

**단점:**
- GPS 오차로 인한 오인증 가능
- 실내에서 GPS 정확도 낮음

#### **C. 티켓 연동 시스템 (권장)**
```
입장권 구매 → 고유 티켓 번호 발급 → 앱에 입력 → 당일 전체 기능 사용
```

**장점:**
- 가장 정확한 방문 인증
- 입장권 시스템과 통합 가능
- 재방문 통계 수집 용이

**구현:**
```python
# 티켓 인증 예시
def verify_ticket(ticket_number, visit_date):
    # 티켓 DB 조회
    if is_valid_ticket(ticket_number, visit_date):
        unlock_full_features(user_id)
        return True
    return False
```

#### **D. 하이브리드 전략 (최종 권장안)**

**무료 기능 (방문 없이 사용 가능):**

- ✅ 운영 정보, 교통 안내
- ✅ 간단한 과학 퀴즈 (5문제)
- ✅ 샘플 오디오북 (30초 미리듣기)

**프리미엄 기능 (방문 인증 필요):**
- 🔒 전체 퀴즈 & 질문 모드
- 🔒 나만의 과학동화 오디오북 (전체 버전)
- 🔒 방문 기록 기반 맞춤형 학습
- 🔒 과학원리 심화 학습

**재방문 인센티브:**
- 🎁 3회 방문 시: 특별 오디오북 (과학자 이야기)

- 🎁 10회 방문 시: 프리미엄 평생 이용권

### **사용 시나리오별 비용 계산**

#### **시나리오 1: 일반 챗봇 대화 (1회)**
```
- 사용자 질문: 50 tokens
- RAG 컨텍스트: 500 tokens
- LLM 답변: 300 tokens
- 모델: GPT-4o-mini

비용 = (550 * $0.15 + 300 * $0.60) / 1,000,000
     = $0.00026 (약 0.3원)
```

#### **시나리오 2: 음성 질문 + 음성 답변 (1회)**
```
- 음성 입력 (30초): $0.006 / 60 * 30 = $0.003
- 챗봇 대화: $0.00026
- TTS 출력 (300자): 300 * $15 / 1,000,000 = $0.0045

총 비용 = $0.00776 (약 10원)
```

#### **시나리오 3: 오디오북 생성 (1회)**
```
- 동화 생성 (GPT-4o):
  입력: 1,000 tokens * $2.50 / 1M = $0.0025
  출력: 1,500 tokens * $10 / 1M = $0.015
  
- TTS-HD (1,500자): 1,500 * $30 / 1M = $0.045

총 비용 = $0.0625 (약 85원)
```

## 3️⃣ **비용 절감 전략**

### **A. 캐싱 시스템 도입**
```python
# 자주 묻는 질문 캐싱
FAQ_CACHE = {
    "운영시간": "화~일요일 09:30~17:30...",
    "입장료": "무료 (단, 일부 프로그램 유료)",
    # ...
}

# API 호출 전 캐시 확인
if question in FAQ_CACHE:
    return FAQ_CACHE[question]  # API 비용 0원
```

**절감 효과:** 약 30-40% 비용 절감

---

### **B. 모델 최적화**
- 간단한 질문: GPT-4o-mini (현재)
- 복잡한 질문/오디오북: GPT-4o
- 티어 구분으로 **20-30% 비용 절감**

---

### **C. 오디오북 사전 생성**
```
인기 놀이터 조합 (AI+탐구, 관찰+행동 등) 
→ 미리 오디오북 생성 (10개)
→ 사용자 요청 시 즉시 제공
```

**절감 효과:** 오디오북 비용 **70-80% 절감**

---

### **D. 음성 압축 및 최적화**
- TTS 속도 조절 (0.9x → 1.0x): 10% 비용 절감
- 음성 길이 최적화 (1,500자 → 1,200자): 20% 절감

## 4️⃣ **예상 문제 및 해결책**

### **문제 1: API 요청 폭주 (DDoS)**
**해결:**
- Rate Limiting: 사용자당 시간당 10회 제한
- 큐 시스템: 요청 대기열 관리

### **문제 2: 오디오북 생성 시간 (1분)**
**해결:**
- 백그라운드 작업: 생성 중 다른 기능 사용 가능
- 진행 상황 표시: "70% 완성..."

### **문제 3: 음성 인식 오류 (시끄러운 환경)**
**해결:**
- 노이즈 캔슬링 전처리
- 재녹음 옵션 제공
- 텍스트 입력 대체 수단

### **문제 4: 서버 다운**
**해결:**
- 오프라인 모드: 기본 안내 정보 캐싱
- 에러 핸들링: 친절한 오류 메시지
